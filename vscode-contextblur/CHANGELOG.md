# Changelog

## 3.1.0 - 2026-03-04

- Added **Go Live Mode** command to enable auto-blur and run a full safety scan in one step.
- Added **Risk Scan** command with proactive warning nudges when critical secrets are detected.
- Improved status bar risk visibility with warning/critical states.
- Updated keybindings to more reliable defaults (`Ctrl/Cmd+Alt+...`) while keeping legacy shortcuts.

## 3.0.0 - 2026-03-04

- Major release preparation for Marketplace publish.
- Standardized packaging workflow with `@vscode/vsce` scripts.
- Added release metadata and Marketplace-ready project files.
- Improved repository hygiene (`.gitignore`, `LICENSE`, `CHANGELOG`).

## 1.0.0

- Initial release of ContextBlur for VS Code.
- Auto-blur scanning for PII and common secret patterns.
- Manual blur command for selected text.
- Status bar integration and style configuration.
