Privacy Policy

ContextBlur does not collect, transmit, or share any personal data. All processing happens locally in your browser.

Data Collection

No personal information, browsing history, or user behavior data is collected. ContextBlur has no servers that receive your data, no analytics pipelines, and no advertising integrations.

Local Storage

ContextBlur stores CSS selectors in chrome.storage.local, keyed by page URL. This data:

  • Never leaves your browser
  • Is not transmitted to any server
  • Can be cleared at any time
  • Contains only CSS selectors (no page content, no personal information)

Auto-blur Feature

The auto-blur feature scans visible text locally in your browser to detect emails, phone numbers, Social Security numbers, and credit card numbers. This process:

  • Happens entirely in your browser
  • Only runs when you click the auto-blur button
  • Requires disclosure acceptance before first use
  • Does not store or transmit any detected content

Payment Processing

Pro subscriptions are handled through ExtensionPay, which uses Stripe for secure payment processing. ContextBlur does not store or have access to your payment details.

Third-Party Services

ContextBlur does not use any analytics, advertising, or tracking services. ExtensionPay is the only external service, used solely for subscription management.

Network Requests

ContextBlur makes minimal network requests, limited to ExtensionPay license verification. No browsing data, page content, or personal information is ever transmitted.

Your Rights

You can clear all stored data at any time by using the "Clear All" button in the side panel, or by removing the extension entirely. No data persists on any external server because no data is ever sent to one.

Contact

If you have questions about this privacy policy, please open an issue on our GitHub repository.
